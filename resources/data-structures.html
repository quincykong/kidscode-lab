Lists, Tuples, Sets, and Dictionaries

LLists [ ]

What: Ordered, changeable collections

When: Store multiple items you might change

# Creating a list fruits = ["apple", "banana", "orange"] numbers = [1, 2, 3, 4, 5] # Common operations fruits.append("grape") # Add item print(fruits[0]) # Get first item fruits[1] = "mango" # Change item

TTuples ( )

What: Ordered, unchangeable collections

When: Store data that won't change

# Creating a tuple coordinates = (10, 20) colors = ("red", "green", "blue") # Common operations print(coordinates[0]) # Get first item print(len(colors)) # Get length # colors[0] = "yellow" # ERROR! Can't change

SSets { }

What: Unordered, unique items only

When: Remove duplicates, check membership

# Creating a set unique_numbers = {1, 2, 3, 3, 4} # {1, 2, 3, 4} animals = {"cat", "dog", "bird"} # Common operations animals.add("fish") # Add item print("cat" in animals) # Check if exists animals.remove("dog") # Remove item

DDictionaries { : }

What: Key-value pairs, like a real dictionary

When: Connect related information

# Creating a dictionary student = { "name": "Alice", "age": 13, "grade": "7th" } # Common operations print(student["name"]) # Get value student["hobby"] = "coding" # Add new pair print(student.keys()) # Get all keys

üîß Common Operations

üìã Lists Operations

# Append (add to end) fruits = ["apple", "banana"] fruits.append("orange") # Result: ["apple", "banana", "orange"] # Remove fruits.remove("banana") # Result: ["apple", "orange"] # Sort fruits.sort() # Result: ["apple", "orange"] # Find (check if exists) if "apple" in fruits: print("Found!") # Iteration (loop through all items) for fruit in fruits: print(fruit)

üì¶ Tuples Operations

# Append - NOT POSSIBLE! # coordinates.append(30) # ERROR! # Remove - NOT POSSIBLE! # coordinates.remove(10) # ERROR! # Sort - NOT POSSIBLE! # coordinates.sort() # ERROR! # Find (check if exists) coordinates = (10, 20) if 10 in coordinates: print("Found!") # Iteration (loop through all items) for coord in coordinates: print(coord)

üîó Sets Operations

# Append (use add instead) animals = {"cat", "dog"} animals.add("bird") # Result: {"cat", "dog", "bird"} # Remove animals.remove("dog") # Result: {"cat", "bird"} # Sort - NOT DIRECTLY! # Use: sorted(animals) returns list sorted_animals = sorted(animals) # Find (check if exists) if "cat" in animals: print("Found!") # Iteration (loop through all items) for animal in animals: print(animal)

üìö Dictionaries Operations

# Append (add new key-value) student = {"name": "Alice", "age": 13} student["grade"] = "7th" # Result: {"name": "Alice", "age": 13, "grade": "7th"} # Remove (delete key) del student["age"] # Result: {"name": "Alice", "grade": "7th"} # Sort - Sort by keys sorted_dict = dict(sorted(student.items())) # Find (check if key exists) if "name" in student: print("Found!") # Iteration (loop through keys) for key in student: print(key, student[key]) # Or loop through key-value pairs for key, value in student.items(): print(key, value)

‚ö†Ô∏è Quick Tips

  • Lists: Use append() to add, remove() to delete, sort() to order
  • Tuples: Can't change after creation! Only use in to find items and for to iterate
  • Sets: Use add() instead of append, sorted() returns a list
  • Dictionaries: Use del to remove, sort by keys or values, iterate with .items()
